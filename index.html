Exercise for M&S Advance Academy

Exercise 1: Structuring Content with HTML Headings

Create an HTML document that uses headings to represent the following hierarchy:

  • M&S Advance Academy (Main Title - <h1>)
  • Principal: Mr. Ibrahim (<h2>)
  • Assistant Principal: Ms. Aminu (<h3>)
  • Head of Departments: Mr. White (<h3>)
  • Teacher: Mrs. Amina (<h2>)
  • Math Teacher: Mr. Adewale (<h3>)
  • Science Teacher: Ms. Blue (<h3>)
  • Student: Abdulmalik (<h2>)
    • Grade 10 (<h3>)
    • Grade 11 (<h3>)
  • Student: Hafsat (<h2>)
    • Grade 10 (<h3>)
    • Grade 11 (<h3>)

Objective: Ensure the correct use of HTML headings to represent a structured hierarchy.

Exercise 2: Working with HTML Entities

Create an HTML paragraph that displays the following:

  • "5 < 10"
  • "Use & to represent < and > symbols."
  • "© 2024 M&S Advance Academy"

Objective: Use HTML entities to display special characters correctly in the browser.

Exercise 3: Creating Hyperlinks in HTML

Create an HTML document with a hyperlink that links to "https://www.greenfieldhigh.edu" with the text "Visit M&S Advance Academy".

Style the hyperlink with CSS to have a dark red color (#4C0001) and no underline by default. Add an underline when the user hovers over the link.

Objective: Practice creating hyperlinks and applying basic CSS styling.

Exercise 4: Embedding and Styling Images

Add an image to your HTML document using the <img> tag. Use "school_logo.png" as the image source and "M&S Advance Academy" as the alt text. Set the image width to 200 pixels.

Apply the following CSS styles to the image:

  • Add a solid border with the color #4C0001.
  • Round the corners with a border-radius of 10px.
  • Ensure the image does not exceed the container's width.

Objective: Learn how to embed images and style them using CSS.

Exercise 5: Using Prettier in VS Code

Write some poorly formatted HTML code and use Prettier in VS Code to automatically format it.

Write an HTML code snippet in one line and use the "Wrap with Abbreviation" feature to properly format it.

Objective: Practice using Prettier for formatting and the "Wrap with Abbreviation" feature in VS Code.
